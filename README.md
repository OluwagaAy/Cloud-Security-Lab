## 🚀 Quick Start

### Prerequisites
- Azure subscription (free tier works)
- OR AWS free tier account
- Terraform >= 1.5
- Azure CLI or AWS CLI

### Setup
```bash
git clone https://github.com/OluwagaAy/Cloud-Security-Lab.git
cd Cloud-Security-Lab

# For Azure
az login

# For AWS
aws configure
```
